Getting Started

How to Enable & Use Cheats

Console commands in Schedule 1 are built into the game on PC. You don’t need mods—just enable the console and start typing.

Step 1 — Enable Console Commands

  • 1 Press Esc to open the pause menu.
  • 2 Go to Settings → Game.
  • 3 Turn on the Console Commands toggle.

Step 2 — Open the Console

  • 1 Exit the menu back into your save.
  • 2 Press the ~ (tilde) key — usually above Tab and left of 1.
  • 3 The console bar appears at the top of the screen.

Step 3 — Enter Commands

  • 1 Type a command exactly as shown in the list on the left.
  • 2 Replace placeholders like <amount> or <item> with real values.
  • 3 Press Enter to run it.
Note: Cheats can change your save permanently. Use a backup save if you’re just experimenting, and be careful with commands that clear inventory, reset quests, or change variables.

Reading the Syntax

• <angle brackets> mean “you choose the value”.
• If no value is shown, just type the command name (for example: freecam).
• Some commands only work in certain locations or when you’re holding a specific item.
